# nodejs quizzes
Wanna play a quiz?


## Dev notes
# Preinstall
* [git](https://git-scm.com/downloads)
* [npm](https://nodejs.org/en/download/)

# Get repo
* git clone https://github.com/sylwolveryn/nodejs-quizzes.git

# Install packages
* cd nodejs-quizzes
* npm install
* npm install mocha -g

# Run tests
* npm test
OR
* mocha

# Start server
* npm start
